# 🍽️ Restaurant Tips Analysis Dashboard

This project was developed by **Coding Hub Students** as part of the **Python Data Analysis Training Program**.

The dashboard analyzes the famous **Tips Dataset** from Seaborn and provides interactive data exploration, filtering, statistical analysis, and visualizations through a Streamlit application.

Live Demo: https://ixeadw6dt8nbb3xjzsmggq.streamlit.app/  
---

## 📊 Dataset

**Tips Dataset (Seaborn Built-in Dataset)**

The dataset contains restaurant customer information such as:

- Total Bill
- Tip
- Gender
- Smoker
- Day
- Time
- Party Size
